HTML Heading

Heading adalah elemen yang digunakan untuk memberikan judul pada halaman web. Ada 6 level heading dalam HTML, dari h1 hingga h6.

Contoh penggunaan heading:
<h1>Judul Utama</h1>
<h2>Sub Judul 1</h2>
<h3>Sub Judul 2</h3>
menjadi heading seperti ini:

Satria Tampan

Sub Judul 1

Sub Judul 2

Heading h1 biasanya digunakan untuk judul utama halaman, sedangkan h2 hingga h6 digunakan untuk sub-judul.

HTML Paragraph

Paragraph adalah elemen yang digunakan untuk menampilkan teks dalam bentuk paragraf. Dalam HTML, kita menggunakan tag <p> untuk membuat paragraf.

Contoh penggunaan paragraph:
<p>Ini adalah paragraf pertama.</p>
<p>Ini adalah paragraf kedua.</p>
menjadi paragraf seperti ini:

Ini adalah paragraf pertama.

Ini adalah paragraf kedua.

HTML Tables

Tabel digunakan untuk menampilkan data dalam bentuk tabel. Contoh:

1. <table> untuk membuat sebuah table
2. <th> untuk membuat header tabel
3. <td> untuk membuat sel tabel
4. <tr> untuk membuat baris tabel

<table>
<tr>
<th>Nama</th>
<th>Umur</th>
</tr>
<tr>
<td>Satria</td>
<td>20</td>
</tr>

Nama Umur
Satria 20

HTML List

List digunakan untuk menampilkan daftar. Contoh:

1. Ordered List:

<ol>
<li>Item 1</li>
<li>Item 2</li>
</ol>

2. Unordered List:

<ul>
<li>Item A</li>
<li>Item B</li>
</ul>

3. Other List:

<dl>
<dt>Definisi 1</dt>
<dd>Deskripsi untuk definisi 1.</dd>
<dt>Definisi 2</dt>
<dt>Deskripsi untuk definisi 2.</dd>
</dl>

1. Ordered List:

  1. Item 1
  2. Item 2

2. Unordered List:

  • Item A
  • Item B

3. Other List:

Definisi 1
Deskripsi untuk definisi 1.
Definisi 2
Deskripsi untuk definisi 2.
